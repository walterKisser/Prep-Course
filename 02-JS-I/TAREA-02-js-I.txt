declaraciones if
¿Qué es la sentencia if?
El condicional if es un comando que nos permite ejecutar una secuencia de comandos
 dependiendo de la condición especificada. Con la misma podemos controlar la ejecución 
de un código en base a las necesidades establecidas por los condicionantes. 
¿Qué es IF y para qué sirve?
La sentencia condicional if se usa para tomar decisiones, 
este evaluá básicamente una operación lógica, es decir una expresión que de como resultado True o False , 
y ejecuta la pieza de código siguiente siempre y cuando el resultado sea verdadero.

Dicho otra forma if es como decir si pasa esto cumple esto sino pasa lo otro(si se cumple la condicion muestra el resultado sin pasa esto.(otro resultado)).

valores booleanos (true,false)

Una valor booleano representa un valor de verdad; es decir, TRUE o FALSE. 
Una expresión booleana o un predicado booleano puede generar un valor de desconocido, 
que se representa mediante el valor nulo.
